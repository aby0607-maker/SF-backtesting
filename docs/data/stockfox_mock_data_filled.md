## Design Philosophy

This mock data is designed with the following first principles:

1. **Real-world authenticity**: Verdicts reflect how Growth, Value, and Beginner investors would actually view each stock
2. **Differentiation showcase**: Same stock, genuinely different verdicts to demonstrate personalization power
3. **Investor education**: Signals and concerns align with actual investment thesis frameworks
4. **Dramatic contrast**: Chosen stocks provide clear demonstration cases (Zomato is polarizing, TCS is consensus quality)
