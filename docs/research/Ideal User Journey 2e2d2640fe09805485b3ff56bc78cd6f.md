# StockFox User Journey - Visual Flow

`USER OPENS APP
    ↓
ENTRY: Sync Portfolio OR Search Stock
    ↓
PHASE 1 (10 min): Analyze 1 stock → Get Overall Score (8.7/10) + Peer Rank (#2 of 12) + 6D Personalization'd **Verdict** (Fits YOUR risk/timeline/experience) + Evidence (94% citations)
    ↓
DECISION: Reject (red flags) OR Proceed (thesis valid)
    ↓
PHASE 2 (Days/Weeks): Analyze 3-5 stocks → Journal tracks decisions → Patterns emerge ("You favor profitable growers") → Portfolio alerts ("70% tech, diversify") → Confidence builds (6/10 → 8.2/10)
    ↓
PHASE 3 (Hours/Days): VALIDATE thesis via Simulator (backtest 5 years, free) OR Advisor (50+ experts, ₹500-15K) OR Execute directly (high conviction)
    ↓
DEPLOY CAPITAL: Auto-setup position tracking + thesis-based alerts
    ↓
REPEAT LOOP: Quarterly earnings check → Score drop alerts → New opportunities → Portfolio rebalancing → Personal changes → RE-ENTER journey

OUTCOME: Confidence 6/10 → 9/10 | Time 45 min → 10 min | Independent decision-maker
MOAT: Only platform with Stock Research & Verdict that is Complete (11-20 segments with 200+ metrics) + Clear (peer rank) + Personalized (6D) + Transparent (citations) + Simple & Fast (AI + Human)`
