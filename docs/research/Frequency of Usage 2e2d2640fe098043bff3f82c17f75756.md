The REPEAT LOOP is where monetization scales:

Initial Journey (Weeks 1-4):
  • Users are in Phase 1-2-3 (learning, building conviction)
  • High engagement but episodic
  • Monetization: ₹99 × 3-5 stocks = ₹297-495

Repeat Loop (Months 2+):
  • Users triggered back by earnings, alerts, capital, opportunities
  • Lower per-session engagement BUT higher frequency over time
  • Monetization: ₹199/month subscription (better value for 3+ triggers/month)

The triggers create HABIT FORMATION without requiring daily usage:
  ✓ Earnings: Predictable quarterly ritual
  ✓ Alerts: Urgent, portfolio-specific
  ✓ Capital: Natural investment rhythm
  ✓ Opportunities: Organic, social-driven

This is why portfolio sync (Phase 2) is critical:
  • Automatic monitoring of holdings
  • Triggers fire based on YOUR stocks, not generic
  • Creates personalized engagement loop
  • Users don't need to remember to check - system alerts them

Unlike competitors:
  • Screener: No triggers, users must remember to check (low retention)
  • Liquide: Generic alerts, not portfolio-specific (noisy)
  • StockFox: Smart triggers based on thesis-breaking events (high signal)

Result: 70%+ Week-4 retention vs industry average 30-40%
